# Prerrequisites

```console
$ sudo pacman --needed -Syu yarn
$ git clone --filter=blob:none --depth=1 git@github.com:cpp-review-dune/cpp-review-dune.github.io.git
$ cd cpp-review-dune.github.io/docs
$ yarn install
$ yarn run dev
```
