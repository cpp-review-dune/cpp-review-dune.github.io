# Python para computación científica 🚀open in new window

# Bienvenido(a) al taller de Python para computación científica

Este taller tiene los siguientes objetivos:

  • Practicar la implementación de funciones y clases en la resolución de una lista de ejercicios.
  • Al término del taller, estará listo para explorar otros módulos de Python como DUNEopen in new window, FEniCSopen in new window o deal.IIopen in new window para la resolución numérica de ecuaciones diferenciales.

# Fechas 📅

  • 06.11.2022
  • 13.11.2022
  • 20.11.2022
  • 27.11.2022

# Prerrequisitos

Para los ejercicios, necesitarás de

  • un navegador webopen in new window para ingresar a Google Colaboratoryopen in new window,
  • un cliente de Zoom Meetingsopen in new window para atender.

Opcionalmente, pero recomendado

  • una cuenta en GitHub para GitHub Discussionsopen in new window,
  • un cliente de mensajería instantáneaopen in new window como Telegram Desktop para preguntar en el canal del taller.
